# AVELLIN

A premium fashion and skincare platform built specifically for the Nigerian market.
The design language combines Jumia's mobile-first practicality with Gucci's premium aesthetic and Shopify's conversion-optimized UX patterns.

## Tech Stack
- **Frontend:** React 18, TypeScript, Vite, Zustand, React Router v6, TailwindCSS, Framer Motion
- **Backend:** Node.js, Express, Supabase

## Setup

1. Clone the repository
2. Install dependencies for both frontend and backend
3. Configure `.env` variables
4. Run frontend (`npm run dev` in `/frontend`) and backend (`node src/index.js` in `/backend`)
